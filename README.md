# Hardop Information Website

This website was created as part of the interdisciplinary student project **"Een Slimme en Sociale Stad"**. Its purpose is to provide social workers, teachers, and other professionals with information about the audio-supported reading books developed by Hardop for low-literacy parents.

The website highlights the mission of Hardop and introduces the audio-assisted storybook *"Nora heeft oorpijn"*. These books are designed to help parents who struggle with reading confidently share reading moments with their children through integrated audio support.

## Technologies

This project was built using:

* HTML5
* CSS3
